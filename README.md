# Record Keeper

Record Keeper is a web app to help vinyl collectors track, manage, and share their collections easily.  

## Features (MVP)
- Add vinyl records manually
- Import collection via CSV
- View collection in a dashboard
- Share read-only view with friends or family

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend & Auth: Supabase
- Hosting: Vercel (frontend), Supabase (backend)
